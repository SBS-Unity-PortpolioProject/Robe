
# SBS 김혜린 포트폴리오

## 프로젝트 소개

#### 1. 프로젝트 명칭 : Robe (2D 플랫포머)
#### 2. 프로젝트 기간 
+ **프로젝트 개발 기간** : 2025년 1월 4일 ~ 2025년 2월 15일(42시간)
+ **프로젝트 개발 목적** : "Sheepy", "Ori and the Blind Forest"를 레퍼런스로 사용하여 학생들이 실제 게임 개발 과정을 체험하고, 시장에 출시 가능한 게임 제작 경험을 제공합니다.
#### 3. 프로젝트 개발 내용
+ **게임 설계** : 게임의 기본 구조, 메커닉, 캐릭터 설정 등 종합적인 게임 설계를 수행합니다.
+ **기술 개발** : Unity 엔진을 사용한 게임 개발, 코드 작성 등이 포함합니다.
+ **아트 및 사운드** : 게임 아트워크, 애니메이션, 사운드 이펙트 및 배경 음악의 제작과 통합을 진행합니다.
#### 4. 사용 기술 및 도구
+ **개발 플랫폼** : Unity
+ **버전 관리** : Git, GItHub
+ **에셋** : [에셋](https://o-lobster.itch.io/platformmetroidvania-pixel-art-asset-pack)
